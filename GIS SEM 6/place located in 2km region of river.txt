Using spatial query find the places those are approximately located within 2 KM region of 
river. 
Show locations from India region
----------------------------------------------------------------------------------------------------------

1) go to Layer  >  Add Layer  >  Add Vector Layer  >  ne_10m_populated_places_simple.shp      and
 								>  ne_10m_rivers_lake_centerlines.shp


2) open Project  >  Properties   >  CRS  >  "World_Azimuthal_Equidistant EPSG 54032"

3) Vector  >  Geoprocessing Tool  >  Buffer  > 
								input layer - ne_10m_rivers_lake_centerlines.shp
								distance  -  0.02     km
								save file  -  river_buffer
	> press Run

go to properties and make label to singleline

6) go to Vector >  Research Tool  >  Select By Location

	select feature from  -  river_buffered
	check  -  intersect column
	comparing to  -  ne_10m_populated_places_simple


Output will be highlighted in yellow colour.
